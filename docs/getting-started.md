# Getting started

### Requirements

- Python 3.5+
- Wagtail 2.3+ and Django 2.0+


### Installation

Install the library:

```
$ pip install wagtailleafletwidget
```


### Quick Setup

Add `wagtailleafletwidget` to your `INSTALLED_APPS` in Django settings.

```python
INSTALLED_APPS = (
    # ...
    'wagtailleafletwidget',
)
```
